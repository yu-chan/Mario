1. 開発環境

OS       : Windows7/10
(ゲーム作成したときの環境に7、10を使用していた。
もしかしたら8でもいけるかもしれない。)
OSビット : 64ビット
IDE      : Visual Studio Express 2012 for Windows Desktop


2. DXライブラリのダウンロード

ゲーム制作には、DXライブラリを用いた。
DXライブラリは下記サイトからダウンロードできる。
http://dxlib.o.oo7.jp/dxdload.html

DXライブラリにはIDE毎に用意されている。
私はIDEにVisual Studioを利用したので、Visual Studio用をダウンロードした。
ZIP形式なので、解凍する必要がある。


3. DXライブラリを使用するためのプロジェクトの設定

DXライブラリをVisual Studioで利用するためにはプロジェクトの設定を行う。
設定については、下記のサイトの「4.DXライブラリを使用するためのプロジェクトの設定を行う」を見ればわかる。
http://dxlib.o.oo7.jp/use/dxuse_vc2012express.html

私が作成したプロジェクトはサイトの内容に従っているが、一つだけ違う点がある。
それは、「4.DXライブラリを使用するためのプロジェクトの設定を行う」の⑥で、パスを「C:\DxLib」にしていることである。
